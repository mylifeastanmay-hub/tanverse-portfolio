# Photoreal Asset Sourcing

Current implementation uses procedural/local geometry and physically based materials, so no external photoreal assets are bundled yet.

Recommended licensed sources for the next fidelity pass:

- HDRI sky lighting: Poly Haven, CC0 HDRIs.
- Terrain PBR texture sets: ambientCG or Poly Haven, CC0 rock/grass/moss/dirt materials.
- Mountain terrain: SRTM/DEM heightmap or licensed photogrammetry terrain tile.
- Bird: licensed transparent video footage or Sketchfab rigged bird model with commercial-use license.
- Purple fluid: pre-rendered licensed fluid simulation video or custom generated fluid texture sequence.

Before commercial deployment, record each chosen asset here with:

- Source URL
- License
- Author/attribution requirement
- File size
- Runtime use location
